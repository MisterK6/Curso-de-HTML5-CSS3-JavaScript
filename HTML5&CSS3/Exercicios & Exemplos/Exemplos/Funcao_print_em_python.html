A função print em Python

A função "print" em Python é utilizada para exibir valores na tela do console. É uma das funções mais comuns e úteis da linguagem Python, e pode ser usada para imprimir uma grande variedade de tipos de dados.
A sintaxe básica da função "print" é a seguinte:

print(valor1, valor2, ..., sep=' ', end='\n', file=sys.stdout, flush=False)

Os argumentos da função "print" são os seguintes:


    * "valor1, valor2, ..." : são os valores que serão impressos na tela.
    Podem ser passados um ou mais valores separados por vírgula. Cada valor será convertido em uma string antes de ser
    impresso.

    * "sep=''" : é o separador que será utilizado para separar os valores que
    estão sendo impressos. Por padrão, é utilizado um espaço em branco (' '), mas pode ser alterado para qualquer outro
    caractere ou string.

    * "end='\n'" : é o caractere ou string que será adicionado ao final da
    impressão dos valores. Por padrão, é utilizado uma nova linha ('\n'), mas pode ser alterado para qualquer outro
    caractere ou string.

    * "file=sys.stdout" : é o arquivo onde os valores serão escritos. Por
    padrão, é utilizado o stdout (a saída padrão do console), mas pode ser alterado para qualquer outro arquivo aberto
    para escrita.

    * "flush=False" : se for True, força a saída a ser imediatamente gravada
    no arquivo, em vez de ser armazenada em um buffer. Por padrão, é False.
